📡 HOW TO SET UP LIVE DATABASE

Do this once — then everyone sees live scores automatically. Free forever.

  1. Go to jsonbin.io → Sign up free (use Google login)
  2. Click + New Bin → paste exactly this as content:
    {"matches":[]}
  3. Click Save → then click the ⚙ Settings icon on your bin → toggle Make Public ✅ — this is essential!
  4. Copy the BIN ID from the URL bar — it looks like 6619abc123def456
  5. Go to API Keys tab on JSONbin → copy your X-Master-Key
  6. Open the downloaded predict-for-today.html in Notepad → find these two lines at the top of the script section and replace the placeholder values:
    const BIN_ID = "your-bin-id-here";
    const API_KEY = "$2a$10$your-key-here";
  7. Save the file → drag & drop it again to app.netlify.com/drop
